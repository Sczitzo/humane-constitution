# Annex Taxonomy — Reader-Facing Schema

## Status and scope

This file is a **reader-facing design artifact**. It is not operative constitutional text, not corpus-registered, and not referenced by any build script.

- Legacy annex IDs (A, B, C … AZ) remain the authoritative file identifiers.
- Decimal codes are display labels for reader navigation only.
- No file renaming is performed by this artifact.
- ANNEX_D is and remains the sole demurrage instrument.
- Draft patches that are not corpus-registered are not implied as adopted by this document.

---

## Design principles

- One decimal code per annex. Each annex has exactly one primary Title assignment.
- Cross-references are notes, not duplicate rows.
- Legacy letter gaps are preserved. They are not filled for neatness.
- Decimal codes organize reader navigation; they do not change constitutional meaning.
- Future migration to INDEX.md or filename aliasing requires separate approval.
