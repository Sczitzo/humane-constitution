# Commons Return and Universal Stake Evidence Test Package

This package defines what must be true before the project can make strong claims about Commons Return and Universal Stake as the live Annex D wealth-and-public-finance instrument.

Commons Return and Universal Stake replaces the retired demurrage spine. The live design no longer depends on idle Flow decay, progressive net-worth demurrage, or demurrage routing to fund public rails. Commons Return captures public value from exclusive control of shared or scarcity-created assets; Universal Stake distributes a universal, non-tradable, non-assignable, non-garnishable member stake that cannot buy Voice, office, survival priority, membership, legal standing, or public favor.

## Claim Under Test

> Commons Return and Universal Stake can return shared/scarcity-created value to the public without converting into a hidden wealth tax on ordinary life, a patronage dividend, a surveillance eligibility system, or a new route for wealth-to-power conversion.

Do not claim yet that:

- Commons Return is proven at constitutional scale;
- Universal Stake distribution rails are capture-proof;
- the source base is sufficient to fund the life-access floor;
- eligibility can be verified without exclusion or surveillance risk;
- valuation hiding, shell entities, external-capital arbitrage, and political routing capture are solved.

## Source Bases To Test

| Source base | What must be measured | Main abuse route |
|---|---|---|
| Land and location value | Site value separate from ordinary homes, improvements, and protected community holdings. | Under-assessment, political exemptions, land trusts as avoidance shells. |
| Natural resources | Extraction value, depletion cost, and public concession terms. | Royalty underpricing, affiliated buyers, reserve misreporting. |
| Spectrum and airspace | License value and scarcity-created monopoly rents. | License warehousing, front bidders, opaque secondary transfers. |
| Monopoly licenses and public concessions | Exclusive grant value and excess return above competitive baseline. | Concession capture, renewal lobbying, procurement favoritism. |
| Platform and network rents | Unavoidable network-position rents where exit is not realistic. | Reclassification as ordinary enterprise income, user-data leverage, transfer pricing. |
| Large succession transfers | Lifetime received-inheritance value above protected thresholds. | Trust chains, valuation discounts, cross-border routing, artificial gifts. |

## Protected Exclusions

The test must prove that ordinary labor income, survival access, basic household exchange, ordinary working balances, ordinary household tools, protected homes below thresholds, and protected community/tribal/church holdings are not treated as Commons Return source bases unless they become avoidance shells.

## Required Tests

### 1. Incidence And Dignity

Measure who actually bears the cost by income, wealth, housing status, business size, locality, and liquidity need. Failure occurs if ordinary households, small stewards, renters, protected associations, or below-threshold homeowners carry the effective burden while high-control actors avoid it.

### 2. Valuation Hiding

Red-team appraisal discounts, affiliated sales, intellectual-property transfers, land-improvement misclassification, mineral reserve understatement, concession accounting, and estate-planning structures. Failure occurs if sophisticated actors can materially reduce liability through complexity that ordinary people cannot access.

### 3. Eligibility Gaming

Test duplicate claims, residency manipulation, household splitting, guardianship misuse, coercive assignment attempts, and exclusion of undocumented, displaced, disabled, digitally fragile, or safety-compromised persons. Failure occurs if Universal Stake eligibility becomes a surveillance gate or excludes vulnerable groups above published thresholds.

### 4. Non-Convertibility Walls

Test whether Universal Stake can be sold, assigned, pledged, garnished, used as collateral, used to buy Voice, used to buy office, used to buy survival priority, used to buy membership, or used to buy public favor. Any successful conversion is a blocking failure.

### 5. Routing Capture

Audit source-by-source public accounting, reserve rules, social wealth fund governance, distribution cadence, administrative discretion, and political messaging. Failure occurs if operators can target rewards or punishments by geography, faction, class, employer, sector, or political compliance.

### 6. External-Capital Arbitrage

Model cross-border ownership, foreign affiliates, treaty shopping, migration of control rights, debt loading, concession resale, and capital inflow timing. Failure occurs if outside capital can hold protected source bases while avoiding the public return or shifting the burden to residents.

### 7. Social Wealth Fund Discipline

If revenue enters a public wealth fund, test passive investment, conflict limits, public audit, draw rules, reserve adequacy, and anti-patronage controls. Failure occurs if fund managers can steer investments, grants, or withdrawals toward political allies or favored sectors.

## Calibration Parameters

The Parameter Calibration Register and founding commitments must track at least:

- source base;
- assessment rate;
- protected thresholds;
- distribution cadence;
- eligibility;
- anti-capture gates;
- fund reserve rules;
- appeal path;
- review cadence.

## Minimum Data Tables

Before scale claims, publish:

1. source-base assessment by source category and protected exclusion;
2. incidence by household type, business type, and locality;
3. Universal Stake distribution by eligibility path and exclusion appeal;
4. valuation adjustment and appeal outcomes;
5. political-routing/capture audit findings;
6. external-capital arbitrage attempts and enforcement outcomes;
7. social wealth fund reserve, draw, and audit data.

## Evidence Threshold

Commons Return and Universal Stake may remain a live designed mechanism only if the tests show that it is:

1. visible to ordinary readers;
2. harder to avoid through wealth, legal complexity, and political influence than the system it replaces;
3. non-convertible into civic power or survival priority;
4. protective of ordinary households and protected associations;
5. administered through public accounting, appeal rights, and anti-patronage controls.

Even if these tests pass, the mechanism remains dangerous. Public return from shared value is still a power to classify, value, collect, and distribute. The goal is not to make that power harmless. The goal is to prove that it is bounded, legible, appealable, and less domination-prone than the concentration it is meant to restrain.
